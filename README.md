# SmartTasker

![SmartTasker Banner](https://via.placeholder.com/1200x300?text=SmartTasker+Task+Manager)

## Overview
SmartTasker is an innovative task management application designed to boost your productivity by intelligently prioritizing your to-dos using AI algorithms. Whether you're managing daily chores or complex projects, SmartTasker helps you focus on what matters most.

## Features
- AI-powered task prioritization
- Intuitive drag-and-drop task organization
- Due date and reminder notifications
- Collaborative task sharing
- Dark mode and customizable themes

## Tech Stack
- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- AI Prioritization: TensorFlow.js
- Authentication: JWT

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/benjaminmind/SmartTasker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SmartTasker
   ```
3. Install dependencies for backend and frontend:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   ```
4. Set up environment variables:
   - Create a `.env` file in the `backend` folder
   - Add variables for your MongoDB URI, JWT secret, etc.
5. Start the backend server:
   ```bash
   cd backend
   npm start
   ```
6. Start the frontend development server:
   ```bash
   cd ../frontend
   npm start
   ```

## Usage
Open your browser and navigate to `http://localhost:3000`. Create an account or log in to start managing and prioritizing your tasks seamlessly.
